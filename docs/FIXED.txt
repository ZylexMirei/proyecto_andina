📋 SI AÚNVES ERRORES:
═══════════════════════════════════════════════════════════════════

  Error: "❌ No hay conexión con Apache/WAMP"
  ────────────────────────────────────────────
  → WAMP no está corriendo
  
  Soluciones:
  1. Verifica que WAMP está instalado en: C:\wamp64
  2. Ejecuta el script diagnose.bat para verificar
  3. Si WAMP no está instalado:
     - Descarga desde: https://wampserver.com
     - Instala en C:\wamp64
     - Reinicia tu computadora
     - Vuelve a ejecutar start-simple.bat


  Error: "npm: command not found"
  ───────────────────────────────
  → Node.js no está instalado
  
  Soluciones:
  1. Descarga Node.js desde: https://nodejs.org
  2. Instala la versión LTS (recomendada)
  3. Reinicia la terminal
  4. Vuelve a ejecutar start-simple.bat


  Error: "ERR_CONNECTION_REFUSED"
  ───────────────────────────────
  → Puerto 3000 está ocupado o WAMP tarda en iniciar
  
  Soluciones:
  1. Cierra todas las terminales
  2. Cierra WAMP completamente
  3. Espera 10 segundos
  4. Ejecuta nuevamente start-simple.bat
  5. Si persiste, cambia el puerto:
     set PORT=3001
     npm run dev
